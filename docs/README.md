# Documentacion

Esta carpeta contiene los documentos de soporte para instalacion, uso, mantenimiento y presentacion formal del software.

## Archivos

- `MANUAL_TECNICO.md`: arquitectura, instalacion, variables de entorno, base de datos, pipeline y API.
- `MANUAL_USUARIO.md`: guia funcional para usuarios finales.
- `REGISTRO_SOFTWARE.md`: checklist y recomendaciones para presentar el repositorio como evidencia de registro de software.

## Uso recomendado

Para evaluacion tecnica, revisar primero `MANUAL_TECNICO.md`.

Para demostracion funcional, revisar `MANUAL_USUARIO.md`.

Para preparar una version oficial en GitHub, revisar `REGISTRO_SOFTWARE.md`.
